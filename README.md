# Suvernica, RH Candidature App


Front :

React-Native

Back : 

Api platform


## Necessite en local 

- Android Studio (emulateur)
- Docker 

## Installation en Local 

- Front: 


```
yarn install 
yarn start
yarn android 
```

- Back :

```
docker-compose build --no-cache
docker-compose up -d

```
